# Imadara Oku Akpabio - Professional Portfolio Website

## Project Overview

A professional, modern, and highly interactive personal portfolio website built with React.js, Tailwind CSS, and Framer Motion. The website showcases both tech expertise and entrepreneurial ventures with a clean, developer-focused design.

## Tech Stack

- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS with custom color system
- **Animations**: Framer Motion
- **Navigation**: React Scroll (smooth scrolling)
- **Icons**: React Icons (Feather, Font Awesome, Simple Icons)
- **Theme**: Dark/Light mode with persistent preference

## Color Scheme

- **Primary**: Royal Blue (#2563EB) with full scale
- **Accent**: Pink (#EC4899) with full scale
- **Navy**: Dark backgrounds (#0F172A)
- **Light Mode**: White + Light Blue gradients
- **Dark Mode**: Navy + Deep Blue backgrounds

## Sections

1. **Navbar** - Fixed, responsive with business dropdown, "Available for Projects" badge
2. **Hero** - Animated intro with role badges, social links, profile placeholder
3. **About** - Full biography, professional summary, animated stats
4. **Skills** - Tabbed layout (Technical/Tools/Soft Skills) with progress bars
5. **Projects** - Interactive cards with hover effects and tech tags
6. **Experience** - Timeline with detail modals for each role
7. **Business** - MAMI Fashion + Tech Services side-by-side layout
8. **Education** - Vertical timeline with animated entries
9. **Certifications** - Cards with detail modal views
10. **Photos & Videos** - Filterable gallery grid
11. **Events** - Cards showing milestones and events
12. **Contact** - Form + contact info + social links
13. **Footer** - Copyright, social links, back-to-top button

## Key Features

- Dark/light mode toggle with localStorage persistence
- Smooth scroll navigation with active section highlighting
- Framer Motion animations on all sections, cards, and interactions
- Responsive design (mobile, tablet, desktop)
- Glass morphism cards and nav effects
- Interactive experience detail modals
- Business dropdown in navbar (MAMI Fashion, Tech Services)
- All social links: WhatsApp, TikTok, Instagram, Facebook, Snapchat, Threads, GitHub, LinkedIn, Twitter
